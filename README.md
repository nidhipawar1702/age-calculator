# Chronos | Premium Age & Life Journey Calculator

Chronos is a visually stunning, premium, and feature-rich Age and Life Journey Calculator. Built with modern glassmorphism aesthetics and smooth micro-interactions, it provides a comprehensive breakdown of a person's life moments.

## Features

- **Exact Calendar Age**: Calculates precise age in Years, Months, and Days.
- **Real-Time Life Ticker**: A live, high-precision ticker showing total years (float), days, hours, minutes, and seconds lived.
- **Next Birthday Countdown**: Counts down the exact months, days, hours, minutes, and seconds to the user's next birthday.
- **Cosmic Profile**:
  - Western Zodiac Sign with Astrological symbols & Elements.
  - Chinese Zodiac Sign with Animal Symbols & Traditional Elements.
  - Numerology Life Path Number calculation and personality description.
- **Life Milestones & Stats**:
  - Birth day of the week.
  - Estimated heartbeats, breaths, and sleep time lived.
  - Interactive clean-number day milestones (e.g. 10,000 Days lived) with exact future dates.
- **Responsive Layout**: Designed for a seamless, luxurious experience on desktop, tablet, and mobile.

## Technology Stack

- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS (Custom properties, keyframe animations, glassmorphism, responsive grids)
- **Logic**: Pure vanilla JavaScript (Precise calendar date differences, high-performance animation frames)

## How to Run Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY_NAME.git
   ```
2. Navigate into the folder:
   ```bash
   cd "age calulator"
   ```
3. Open `index.html` in your web browser, or run a local HTTP server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   ```
4. Visit `http://localhost:8000` (or the port specified by your server).
